Line Break = 4 Methods for Documents

(1) Br Tag = Normal as html
document.write("Hello World!(br tag)Welcome to JavaScript!");
OR
document.write("Hello World!(br tag)")
document.write("Welcome to JavaScript!")
(2) P Tag = Gives more gap btw 2 lines
document.write("(p tag)Hello World!(p tag) Welcome to JavaScript!")
OR
document.write("(p tag)Hello World!(p tag)")
document.write("Welcome to JavaScript!")
(3) Div Tag = Same as br tag
document.write("(div tag)Hello World!(div tag) Welcome to JavaScript!")
OR
document.write("(div tag)Hello World!(div tag)")
document.write("Welcome to JavaScript!")
(4) Pre Tag = Preserves formatting (spaces, line breaks)
document.write("(pre tag)Hello World! \n Welcome to JavaScript! (pre tag)")

Line Break = for Console

(1) \n = New Line Character
document.write("Hello World! \n Welcome to JavaScript!");
OR
document.write("Hello World! \n")
document.write("Welcome to JavaScript!");

Var

Reusable code blocks
var x=10;
var x=20; // Reassigning value to x
var x=30; // Reassigning value to x again
document.write(x); // Output will be 30 (the last assigned value)
var x ="bishesh";
var x="singh"; // Reassigning value to x
var x="chauhan"; // Reassigning value to x again
document.write(x); // Output will be chauhan (the last assigned value)

let

Not Reusable code blocks
let x=10;
let x=20; // Reassigning value to x
let x=30; // Reassigning value to x again
document.write(x); // Output will be blank screen
we can only declare a variable once with let in the same scope. If we try to declare it again, it will throw an error.
let x =10;
x =20; // This will cause an error because x is already declared
x =30; // This will also cause an error
document.write(x); // This will not execute due to the error
same goes for string
let x ="bishesh";
x="singh"; // Reassigning value to x
x="chauhan"; // Reassigning value to x again
document.write(x); // Output will be chauhan (the last assigned value)

const(Constant variable)

Not Reusable code blocks
const x=10;
const x=20; // Reassigning value to x
const x=30; // Reassigning value to x again
document.write(x); // Output will be 10 (the initial assigned value)
we cannot reassign a value to a variable declared with const. If we try to reassign it, it will throw an error.
const x =10;
x =20; // This will cause an error because x is a constant
x =30; // This will also cause an error
document.write(x); // This will not execute due to the error
same goes for string
const x ="bishesh";
x="singh"; // This will cause an error because x is a constant
x="chauhan"; // This will also cause an error
document.write(x); // This will not execute due to the error
only one time we can assign a value to a const variable
