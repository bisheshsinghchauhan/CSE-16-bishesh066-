Methods for javascript type 1

JavaScript provides several built-in methods for handling user interactions and displaying messages. Here are some of the most commonly used methods:

1. Alert Box

The alert() method displays a simple alert dialog with a message and an OK button.

alert("Welcome to my web page");

2. Confirm Box

The confirm() method displays a dialog with a message and two buttons: OK and Cancel. It returns true if the user clicks OK, and false if they click Cancel.

confirm("Do you want to continue?");

3. Prompt Box

The prompt() method displays a dialog with a message, an input field, and two buttons: OK and Cancel. It returns the input value if the user clicks OK, and null if they click Cancel.

prompt("Please enter your name:");
